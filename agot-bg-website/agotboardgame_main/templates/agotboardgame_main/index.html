Welcome to this open beta-test!

At the moment, the focus is to finish implementing the base game and polish the existing features.

The final goal is to implement all the release gameplay of the extensions (Feast for Crows, Dance with Dragons and Mother of Dragons), and add meta-game features (abuse reports, ranked games, ...).

Bugs and suggestions of features can be posted on the Github of the project.

Feedbacks and other remarks can also be posted on the Discord!

What to test

The implementation of the rules of the game is the priority. Feel free to play as you want, but don't hesitate to check if the game reacts correctly to edge cases (special combinations of house cards, for example).
