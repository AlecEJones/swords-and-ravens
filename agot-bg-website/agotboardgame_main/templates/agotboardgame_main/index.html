Welcome to Swords and Ravens!

Swords and Ravens is a platform to play the board game "A Game of Thrones: Board Game - Second Edition", edited by Fantasy Flight Games, online with players around the world!

The game is currently in open beta test in order to iron the features of the base game. Plans are to implement the gameplay of the extensions (especially Mother of Dragons).

Feedbacks, bug reports and other remarks can also be posted on the Discord!

The source code can be found on the Github of the project.

What to test

The implementation of the rules of the game is the priority. Feel free to play as you want, but don't hesitate to check if the game reacts correctly to edge cases (special combinations of house cards, for example).
